# HOMLAP - Modern Housing Solutions

HOMLAP is a premium, zero-brokerage rental housing platform for verified PGs, flats, and rooms across India. Built with speed and reliability in mind, it connects tenants directly with property owners for a seamless living experience.

## Features

- **Zero Brokerage**: Connect directly with owners and skip the fees.
- **100% Verified**: Every listing is physically verified by our team.
- **Modern UI**: A premium, responsive interface for effortless navigation.
- **Nationwide Coverage**: Find homes in Mumbai, Delhi, Bangalore, Pune, and more.
- **Property Varieties**: From luxury flats to student-friendly hostels.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4, Framer Motion
- **Icons**: Lucide React
- **Deployment**: Optimized for Vercel

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
